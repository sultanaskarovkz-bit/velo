# VELO - лендинг велосипедов ASEKA и IBARS

Продающий одностраничный сайт для магазина велосипедов VELO (линейки ASEKA и IBARS).

## Возможности
- Лид-формы (герой и нижний блок) с отправкой заявки в WhatsApp
- Кнопка звонка и липкая мобильная панель (звонок / WhatsApp)
- Каталог моделей из реальных фото
- Адаптив под все устройства (десктоп, планшет, телефон)
- Зелёно-белая тема, SVG-фавикон, SEO и Open Graph

## Запуск
Статический сайт. Откройте `index.html` в браузере или разместите папку на любом хостинге.

### GitHub Pages
Settings -> Pages -> Deploy from branch -> `main` / root.
